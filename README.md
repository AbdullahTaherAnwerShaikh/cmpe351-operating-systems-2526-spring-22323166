# CPU Scheduler Simulator
In this project, it is expected to develop a CPU scheduler application that simulates many
different scheduling algorithms
Inputs must be taken from text file with command arguments. And results also should be
written to output file and screen.

### Before anything
You need to copy the template file according to your programming language preference. For C, 
the cmpe351.c.orig file should be copied as cmpe351.c,and for C++, the cmpe351.cpp.orig file 
should be copied as cmpe351.cpp.

### To compile project
`make all`

### To Test the code
`make test`
